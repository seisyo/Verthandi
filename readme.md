##簡介

Project Verthandi 為台彎學生所聚集而成的研討會 "SITCON 學生計算機年會" 財務組建立之財務系統
（SITCON Financial System）建置計畫。

此系統建置計畫大致分為兩階段：

第一階段為建置伍此財務系統中 “會計功能” 的部分。功能包含會計基本流程之日記簿、分類帳和生成報表。
此階段的使用者限為總副召和財務組組員。

第二階段為開放各組組長可至此系統之ISSUE管理部分，可以將每一筆支出以一個個ISSUE的方式做管理和追蹤。
